# EC2機能
- インスタンス作成・一覧・削除
- SG作成・一覧・削除

# 画面
- インスタンスパネル  
    - 一覧で表示する項目
        - Name  
        - ID  
        - Type  
        - Ava  
        - status  
        - SG  
    - 操作  
        - インスタンス作成  
        - インスタンス削除  
        - インスタンス更新(主に停止）  

- SGパネル  
    - 一覧で表示する項目
        - GroupID  
        - GroupName  
        - VPCID  
    - 操作  
        - 作成  
        - 削除  
        - インバウンド編集  
        - アウトバウンド編集
